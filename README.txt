Ce programme est un jeu de puissance 4:

Chaque joueur possède une couleur, rouge ou jaune.
Le joueur qui jouera en premier est choisi aléatoirement par le programme.
Dans une grille de 7 colonnes chaque jour peut ajouter un jeton en selectionnant la colonne, de manière verticale, tour par tour.
Le premier joueur qui aligne 4 jetons de sa couleur (en diagonale, horizontalement ou verticalement) a gagné la partie.
Les coups de chaque joueur peuvent être bloqués par le joueur suivant, le joueur peut également revenir  en arrière.
Il est possible pour le joueur de recommencer une partie à l'aide du boutton nouvelle partie.
Le joueur peut également sauvegarder une partie pour pouvoir la charger ultérieurement et continuer sur sa progression précedente.
